踏入職場的第一步

Sudo 生存指南
2016,May

實習，夾縫在學生與工作兩種身份之間。說是新人好像太稚嫩，說是實習生又似乎太沒格調，那麼就叫做微菜鳥吧！

剛好 Sudo 最近新進了很多微菜鳥，身為微菜鳥的你到底該如何進入職場呢？如何快速與同事製造互動呢？今天 KK 會帶大家掀開 Sudo 的暗紗，用最真實的案例告訴你該如何與同事親密接觸

本期專欄： Sudo 潛規則大公開

閱讀注意：本篇文章將會揭露最赤裸的 Sudo 潛規則！

踏進 Sudo 的第一步 – 買豆花？

你沒有看錯，就是豆花。

剛進辦公室時，你會發現大家正用詭異的眼神注視著你。眼神裡似乎閃爍著，恩…？狡黠的光芒，那是期待新人到來的眼神嗎？不，更適切的形容應該是狡猾。

「新來的要負責訂豆花哦！」

原來想要進入 Sudo，除了面試之外，還有一道隱藏關卡，就是訂豆花！

用豆花拜碼頭？

這個儀式顯然已經行之有年，連 slack 上都有 bot 紀錄豆花的菜單。Sudo 裡頭的同事對豆花（或者食物）有種近乎瘋狂的熱愛。

work hard、當然也要 eat a lot。

Sudo 的微菜鳥們要接受訂豆花的考驗。

不要小看訂豆花。任何有訂過的人都知道，看似簡單，其實裡頭卻也蘊含許多小細節。

  • 口味的選擇
  • 零錢
  • 掌握正確的數量
  • 甜度冰塊
  • 打電話
  • 分類
  • 各種奇怪的癖好（如：不要加豆花）

不時還會有人插隊或是突然冒出來 +1，或者訂到一半有突然取消等等。要應付的情況還真的挺多的！
這對剛進去的新人來說是一種挑戰，卻也是讓大家留下好印象的機會。

訂豆花，可以不只是豆花

你發現了嗎，原本以為只是同事間為了滿足吃豆花的慾望（事實上好像也是），但從本篇文章中看下來，居然巧妙地成為了與同事互動的好辦法。

趁著詢問同事口味的同時，順便也讓同事了解你。我想，這大概是這個儀式裡隱藏的涵義吧！有了適當的理由（訂豆花）之後，新人與同事之間的互動不再那麼尷尬，也能順利讓彼此留下深刻印象。

尾聲

踏入 Sudo 的第一步是訂豆花。套用到職場上或許行得通也說不定，下次主動喊一聲：「不然我來幫忙訂…吧！」

當你看到同事們感激的眼神，好像也值得了，對吧！

「你今天訂豆花了嗎?」
