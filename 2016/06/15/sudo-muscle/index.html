番外篇 - Sudo 猛男棒式教學

番外篇
猛男
2016,Jun

Sudo 裡頭總是充滿許多挑戰。累積一天的壓力都會頓時在下班後暴走，並且以各式各樣的形式出現。而這一次，讓我們來瞧瞧 Sudo 的下班日常。

「來 PK 棒式啦！」

這一次，挑釁的旌旗是由 RD team 的 leader 發起。
說話帶有工程師特有簡潔的他，竟然難得尋找對手。

約在一年前，顆顆與老闆立下了「人魚線之賭」，賭在一年之後，是否能夠練出人魚線或是馬甲線。看來這次掀起戰火，是有備而來啊！讓我們繼續看下去。

肌肉發達的程度是否跟寫 code 的功力成正比呢？這個疑問先放在心裡頭，答案會在最後揭曉。

本次接受挑戰的有：

顆顆(RD)、Andy(Sales)、Eric(COO)、Peter(Designer)

棒式教學起手式：

  • 背打直
  • 肚子繃緊
  • 屁股夾好
  • 收下巴

基本上大原則有下列四點，訂定好規則之後，遊戲就可以準備開始囉！

5 4 3 2 1 … GO

最後一名

看來鍛鍊的效果可能不夠明顯（笑），這個發起戰帖的人竟然最先投降。同時我們也學到了重要的一課：「寫 code 和練肌肉不能兼得」。

好啦以上純屬玩笑，身為工程師，整天坐在電腦螢幕前，還是應該保有規律運動的習慣，才不會造成血液循環不良及長期的肌肉痠痛。sitting is smoking

記得讓視線與螢幕平行，才不會造成脖子酸痛等問題哦！

在 Sudo 要是低頭看螢幕，CEO 還會跑來關注你，問你要不要買個 24 寸螢幕跟電腦架。而且是真的會買！

倒數第二名

我們的設計師雖然看起來肌肉滿滿，但其實也不過如此嘛！剛開始的時候還故意穿外套加厚身形偷吃步，結果得了便宜也沒有撐比較久。有肌肉並不代表有力氣，看起來 design 跟肌肉好像也不能兩者兼得呢！

第二名

這個人看起來高高瘦瘦，沒想到棒式比賽裡頭他竟然勇奪第二名寶座。

果然應證了寫 code 與肌肉發達程度成反比

第一名

第一名出爐，由我們的 COO Eric 獲勝！在剛開始的幾分鐘幾乎臉不紅氣不喘，還一直對 Andy 吹氣擾亂敵情。雖然有點勝之不武，但也算是實至名歸啦！

背著公司之外還背著家庭，有肩膀的男人果然不一樣

真的很不一樣。

Final Word

這些場景對 Sudo 來講已經算是鳳毛麟角了。舉凡各種光怪陸離的場景，在 Sudo 裡頭都會發生。

這裡不適用標準的職場規則，但歡迎每一位充滿活力的你/妳。
