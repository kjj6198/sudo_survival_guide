踏入職場的第二步跟第三步

這次的篇幅比較短一些，所以跟第三步一起大放送！

第二步 - 接受惡婆婆的攻擊

在 Sudo 有個惡名昭彰的姐姐，平常喜歡欺負人，不害怕公權力(因為她就是公權力)。

而他的其中一項樂趣，就是欺負 RD Team 的實習生。在一來一往的經驗中，我們已經 找到最佳的對應之道:「沈默是金」

從圖中可以看到，我們的設計師實習生把 Sudo 姐姐的邪惡笑容描繪得栩栩如生。
那是她的詭計或計畫得逞時，會露出招牌詭笑，同時將頭上仰 87.87 度。

這個外表看似光鮮亮麗的 Sudo 姐姐其實深藏許多內幕，可以說是人不可貌相的最佳代言人。

不過，這位姐姐也不是只會欺負人而已。

她經手了許多 Sudo 大大小小的活動規劃，熟悉台北各個優質展場，在交涉的過程中還會不著痕跡地將預算殺到最低。但品質永遠超乎你的想像，包括流程、場地、食物、交通…各種細節都已經幫你默默處理好了，剩下的就是盡情享受活動！

想要一探究竟嗎？想要參加 Sudo 姐姐舉辦的各種活動嗎？剛好 Sudo 與 make school 籌辦了 summer academy，趕快來感受一些 Sudo 姐姐的威力吧！

第三步 - 與 Co-founder 一起打網咖

「幾個臭男生聚在一起，就是要打網咖啊，不然要幹嘛？」

咦？怎麼跟想像中不太一樣？

職場上和主管一起跑去打網咖，這種違和的畫面在 Sudo 已經不算稀奇了。Co-founder 們在長期的經營之下，全台各大據點，他們幾乎都有會員。

屈服於權威之下，懵懂無知的我也就這樣踏上了不歸路。

偶爾，閒來無事的 friday night，就是我們的 fighting night。
常常 Andy 或 Eric 一聲令下，Sudo 戰隊就這樣成行了。打過世紀帝國、戰慄時空、星海爭霸……，這些遊戲我們都略懂略懂。

在工作之餘，我們也懂得放鬆。

當然，不只打遊戲而已，它們還懂得健身呢！
